**8. Critical Reflection**

In this homework, we established a machine learning baseline for the Iris dataset using a K-Nearest Neighbors (KNN) model. The algorithm achieved a test accuracy of 96.67% when using an optimal value of 5 neighbors. In this toy setting, "generalizing" means the model correctly classifies new flowers it has never seen before. However, a high accuracy can sometimes be misleading. For example, if there is data leakage, the model just memorizes the test data instead of actually learning. Also, because the Iris dataset only has 150 samples, our test set is very small. This small size can cause an unstable metric, where just one mistake drops the accuracy percentage significantly. Another risk is overfitting to noise. If we had used k=1, the model would strictly memorize random errors, failing to capture the real distribution of the plants. The fact that our accuracy remained high and stable across multiple K values shows that the model is genuinely generalizing rather than just getting lucky. Finally, this KNN approach demonstrates successful generalization on a classic dataset while highlighting the importance of hyperparameter tuning to prevent false results.
